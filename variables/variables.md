# Variables in D 

Similar to C, in D, we first define the variable data type and the variable name 

```d
int studentCount = 0;
double weight = 0.0;
```

## Printing variables 


To print variables, we can use the `write` and `writeln` functions as follows 

```d
writeln("Student Count: ", studentCount);
writeln("Weight: ", weight, " kg");      // TO print it as 28 kg or 45 kg
```
